OPlayer
=======

Android平台基于Vitamio的开源播放器

支持以下协议的音频和视频播放：
* MMS
* RTSP (RTP, SDP)
* HTTP流式传输(progressive streaming)
* HTTP Live Streaming (M3U8), Android 2.1+
Vitamio集成了许多音频和视频的解码包，相比Android内置默认的媒体格式，这里列出其中的一些：
* divx/xvid
* wmv
* flv
* ts
* rmvb
* mkv
* mov
* m4v
* avi
* mp4
* 3gp
